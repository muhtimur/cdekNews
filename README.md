# cdekNews
Маленький сервер для получения новостей

Пример работы

http://1e14c3489fcb.vps.myjino.ru:5000/api/v1/news/feed/company/short?perPage=3&page=1

```bash
# Клонируйте репозиторий с API
git clone https://github.com/muhtimur/cdekNews.git

# Установите зависимости
pip install flask waitress flask_cors

# Запустите сервер
python app.py
```
